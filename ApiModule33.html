API de Consulta do Módulo de Registro Escolar

Documentação completa para desenvolvedores consultarem a API de registros de estudantes e documentos

Laravel 12 PHP 8.4

Autenticação

Para acessar os endpoints protegidos, primeiro obtenha um token de acesso através do endpoint de login.

1. Login
POST /api/login

Autentica o usuário e retorna um token de acesso.

POST /api/login Content-Type: application/json { "email": "admin@example.com", "password": "senha" }
{ "token": "1|abc123def456..." }
2. Usando o Token

Para endpoints autenticados, inclua o token no header Authorization:

Authorization: Bearer 1|abc123def456...

Endpoints de Consulta da API

GET /api/students

Lista paginada de estudantes.

Parâmetros de Query:
  • page (opcional): Número da página (padrão: 1)
  • per_page (opcional): Itens por página (padrão: 10)
GET /api/students?page=1&per_page=10 Authorization: Bearer YOUR_TOKEN
{ "results": [ { "name": "Marcos Silva", "cpf": "99999999999" } ], "paginate": { "currentPage": 1, "totalPages": 10, "total": 95 } }
GET /api/students/{cpf}

Detalhes do estudante com lista de documentos.

Parâmetros de Path:
  • cpf (obrigatório): CPF do estudante
GET /api/students/99999999999 Authorization: Bearer YOUR_TOKEN
{ "name": "Marcos Silva", "cpf": "99999999999", "archives": [ { "id": 1, "name": "CPF", "link": "Y29udGV1ZG9jb2RpZmljYWRvZW1iYXNlNjQ=", "date": "2023-10-30 12:00:00" } ] }
GET /api/students/{cpf}/documents/{id}

Informações detalhadas do documento.

Parâmetros de Path:
  • cpf (obrigatório): CPF do estudante
  • id (obrigatório): ID do documento
GET /api/students/99999999999/documents/1 Authorization: Bearer YOUR_TOKEN
{ "name": "Documento.pdf", "folder": "Aluno X", "category": "Categoria", "type": "Tipo", "date": "31-10-2025 14:00:00" }
GET /api/students/{cpf}/documents/{id}/viewDocument

Visualiza o conteúdo do documento (stream inline).

Parâmetros de Path:
  • cpf (obrigatório): CPF do estudante
  • id (obrigatório): ID do documento
Este endpoint retorna o conteúdo binário do arquivo, não JSON.
GET /api/students/99999999999/documents/1/viewDocument Authorization: Bearer YOUR_TOKEN

Headers de resposta:

  • Content-Type: Tipo MIME do arquivo (ex: application/pdf)
  • Content-Disposition: inline; filename=nome_do_arquivo
